#######
Para executar o projeto

1 - instalação de dependencias:

pip install -r requirements.txt


2 - para subir o projeto:

python manage.py runserver


3 - durante o start do projeto será importado o arquivo movielist.csv


4 - todos os metodos estão disponiveis no endereço raiz


#######
Rodar o teste de integração:

Com o servidor já rodando no endereço 127.0.0.1:8000 deve ser executado o comando:

python manage.py test
